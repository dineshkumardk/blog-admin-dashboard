# Blog Admin Dashboard

## Tech Stack
- React + Vite
- Tailwind CSS
- LocalStorage

## Features
- Admin Dashboard with summary cards
- Blog CRUD operations
- Soft delete with auto purge after 7 days
- Image upload with validation & preview
- Pagination, search & filters
- Responsive admin layout
